Devloped on MacOS using IntelliJ IDEA CE.
Compile and run on Linux machines with the following commands:
1. > javac HW01.java
2. > java HW01

  • "Close" closes the program
  • "Rectangles" switches the program to create rectangles
  • "Tessellated Circles" switches the program to create tessellated polygons the resemble circles
  • "Reset" clears the screen of all objects
  • "Pause/Play" starts and stops the falling effect of the objects
  • "▲ Radius/▼ Radius" increases or decreases the tessellated circles' radius
  • "▲ Tessell/▼ Tessell" increases or decresases the tessellated circles' degree of tessellation
